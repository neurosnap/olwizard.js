olWizard.js

A jQuery step wizard plugin that gets out of the way.

This library utilizes the HTML ordered-list ol tag and li tags to create a wizard-like step functionality.

The primary goal of this library is to facilitate the basic functionality for a wizard without assuming the style of it.

HMTL

Here's the basic HTML structure

<ol id="weezy">
  <li>
    <div class="olwiz-title">First Step</div>
    <div class="olwiz-content">
      This is the first step
      <button type="button" class="olwiz-prev">Prev</button>
      <button type="button" class="olwiz-next">Next</button>
    <li>
  </li>
  <li>
    <div class="olwiz-title">Second Step</div>
    <div class="olwiz-content">
      This is the second step
      <button type="button" class="olwiz-prev">Prev</button>
      <button type="button" class="olwiz-next">Next</button>
    <li>
  </li>
</ol>

The ordered-list structure dictates the flow of the wizard.

Initialize olWizard.js

var weezy = $("#weezy").olWizard();

That's all you need to get started. olWizard.js tries to be as unassuming as possible. No HTML is generated in the library, it's up to the end-developer to generate the HTML and CSS.

Options
Property Default Description
title .olwiz-title Class that corresponds to the step title
content .olwiz-content Class that corresponds to the step body content
next .olwiz-next Class that the next button event handler uses
prev .olwiz-prev Class that the previous button event handler uses
passed .olwiz-passed Class that signifies if the step has passed
failed .olwiz-failed Class that signifies if the step has failed
active .olwiz-active Class that signifies if the step is active
olWizard Methods
Method Arguments Description
gotoStep number | name Navigates to the step number or the name of the step
getStep number | name Returns the lilWizard instance of the step
getActiveStep - Grabs the lilWizard instance of the active step
next lilWizard jQuery object (optional) Goes to the next step
prev lilWizard jQuery object (optional) Goes to the previous step
lilWizard Methods
Method Arguments Description
getStepNumber - Returns the current step number
isFirst - Determines if step is the first step
isLast - Determines if the step is the last step
data object Get/Set data of the step
prev lilWizard jQuery object (optional) Goes to the previous step
passed - Signifies if step has passed validation
failed - Signifies if step has failed validation
validate function Callback that validates the step. It is expecting to return true, false, or undefined. If undefined, validation won't pass until the passed() method is called for the lilWizard instance. If no callback is supplied for a step, the step will automatically pass validation
Events
Event Description
olwizStep Triggers right before a step starts
olwizStepPassed Triggers after step has passed
olwizStepFailed Triggers after step has failed
olwizDone Triggers after all steps are completed
